# 📺 FlickRift  

![FlickRift logo](https://github.com/AMALIK8989/FlickRift/blob/main/Assets/Logo.png)  

**FlickRift** is your go-to streaming platform for free, high-quality movies and TV shows, offering a wide range of content for everyone—with **no login required**. Enjoy unlimited entertainment without subscriptions or sign-ups!  

---

## 🚀 Features  

✅ **Stream Free Movies & TV Shows** – Watch high-quality content anytime.  
✅ **No Sign-Up Required** – Start streaming instantly, no login needed.  
✅ **Fast & Responsive** – Optimized for all devices.  
✅ **User-Friendly UI** – Simple, sleek, and intuitive design.  
✅ **Smooth Animations** – Powered by **Anime.js**, **Animate.css**, and **WOW.js**.  
✅ **Carousel Sliders** – Featuring **Owl Carousel** for smooth navigation.  
✅ **Font Awesome Icons** – Enhanced visual appeal with stylish icons.

## 🎬 Getting Started  

### 🔧 Installation  

1️⃣ **Clone the repository**  

```sh
git clone https://github.com/AMALIK8989/FlickRift.git
```

2️⃣ **Navigate to the project folder**  

```sh
cd FlickRift
```

3️⃣ **Open `index.html` in your browser**  

That's it! Enjoy streaming movies and TV shows on **FlickRift**.
